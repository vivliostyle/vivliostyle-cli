# Packages kept in the image that build/audit.ts's runtime closure cannot reach.
# audit.ts fails the build if a package outside the closure is listed in none of
# keep-packages.txt / purge-packages.txt / purge-packages-late.txt, so each one
# here is an explicit "keep" decision. To re-derive, empty the purge files and
# triage what audit.ts reports -- the same loop as build/essential-packages.txt.
#
# The closure follows DT_NEEDED links, so it cannot see:
#   data       packages with no .so, read via /usr/share/* through the
#              fontconfig / icon / mime caches.
#   essential  not reached from the seed binaries, and used without being
#              declared as a dependency (dpkg shells out to /usr/bin/diff).
#   dlopen     loaded at runtime with no DT_NEEDED entry.
#   exec       spawned as a subprocess, not linked.
