# keep-anyway: packages that are removable (the runtime closure cannot reach them)
# but that the image must keep. The build does NOT read this file; it is input to
# the out-of-band curation tool build/audit-removable.ts, which subtracts it from
# the candidate list so only genuinely-removable, genuinely-unwanted packages are
# reported. Each entry is a blind spot of that closure (lddtree's DT_NEEDED tree of
# the bundled chrome + apt-cache --recurse): a dependency loaded by dlopen, linked
# only by firefox, reached only by exec, or pure data with no .so. To regenerate,
# run audit-removable.ts and move whatever it reports as "unclassified removable"
# here (kept) or into purge.txt (dropped). image-contract.sh is the real proof the
# kept set is sufficient.
