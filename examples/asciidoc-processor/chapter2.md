# Chapter 2: Markdown

This chapter is written in **Markdown** and processed with the default VFM processor.

## Features

Markdown supports many features:

- **Bold** and _italic_ text
- `Monospace` for code
- Links: [Vivliostyle](https://vivliostyle.org)

## Code Example

```javascript
function hello() {
  console.log('Hello from Markdown!');
}
```

## Summary

Markdown files use the default VFM processor without per-entry settings.
