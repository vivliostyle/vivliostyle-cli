# Mixed Format Example (AsciiDoc + Markdown)

This example demonstrates how to mix AsciiDoc and Markdown files in a single publication using per-entry `documentProcessor` and `documentMetadataReader`.

## Project Structure

```
├── chapter1.adoc           # AsciiDoc (custom processor)
├── chapter2.md             # Markdown (default VFM)
└── vivliostyle.config.js
```
